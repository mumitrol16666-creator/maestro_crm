# 🎨 Замена эмодзи на SVG иконки

## Дата: 12 октября 2025, 19:30

## Проблема
В уведомлениях использовались эмодзи (🧊❄️💎📞⚠️❌✅), которые не соответствуют стилю сайта.

## Что сделано

### 1. Создана библиотека SVG иконок
**Файл:** `frontend/js/script.js`

Добавлены функции для работы с иконками:

```javascript
function getIcon(type, size = 20)
function notificationWithIcon(icon, message)
```

#### Доступные иконки:
- `freeze` - снежинка (для заморозки)
- `phone` - телефон
- `warning` - предупреждение (треугольник)
- `error` - ошибка (крестик в круге)
- `success` - успех (галочка в круге)
- `diamond` - бриллиант
- `tool` - инструменты
- `check` - галочка
- `user` - пользователь
- `calendar` - календарь
- `plus` - плюс в круге

### 2. Обновлена функция showNotification
**Файл:** `frontend/js/script.js`

Теперь поддерживает HTML для отображения иконок:
- Автоматически определяет, есть ли в сообщении HTML
- Использует `innerHTML` для сообщений с иконками
- Использует `textContent` для обычного текста

### 3. Заменены все эмодзи в profile.js

#### До:
```javascript
showNotification('Заморожено занятий: 1 🧊');
showNotification('❄️ Все доступные заморозки уже использованы.\n\n...');
showNotification('💎 Для продления абонемента...');
```

#### После:
```javascript
showNotification(notificationWithIcon('freeze', `
    Заморожено занятий: <strong>1</strong>
`));

showNotification(notificationWithIcon('freeze', `
    Все доступные заморозки уже использованы.<br><br>
    Для дополнительных заморозок обратитесь к администратору:<br>
    <strong>+7 (700) 095-09-04</strong>
`));

showNotification(notificationWithIcon('diamond', `
    Для продления абонемента обратитесь к администратору:<br>
    <strong>+7 (700) 095-09-04</strong>
`));
```

### 4. Обновлены все уведомления

| Действие | Иконка | Эмодзи до |
|----------|--------|-----------|
| Успешная заморозка | `freeze` | 🧊 |
| Нет заморозок | `freeze` | ❄️ |
| Предупреждение | `warning` | ⚠️ |
| Ошибка | `error` | ❌ |
| Продление абонемента | `diamond` | 💎 |
| Редактирование | `tool` | - |

## Формат уведомлений

### С иконкой слева:
```
┌────────────────────────────────┐
│  [иконка]  Текст сообщения     │
│            Продолжение текста  │
│            Телефон: +7 700...  │
└────────────────────────────────┘
```

### Структура:
- Иконка слева, розового цвета
- Текст справа с отступом
- Многострочный текст поддерживается
- HTML форматирование (`<br>`, `<strong>`)

## Использование

### Простое уведомление:
```javascript
showNotification('Обычное текстовое сообщение');
```

### Уведомление с иконкой:
```javascript
showNotification(notificationWithIcon('success', `
    Операция выполнена успешно!<br>
    <strong>Все данные сохранены</strong>
`));
```

### Добавить новую иконку:
1. Откройте `frontend/js/script.js`
2. Найдите функцию `getIcon`
3. Добавьте новую иконку в объект `icons`:
```javascript
myIcon: `<svg width="${size}" height="${size}" ...>
    <path d="..."/>
</svg>`
```
