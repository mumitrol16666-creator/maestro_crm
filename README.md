## 💰 БИЗНЕС-ЛОГИКА АБОНЕМЕНТОВ

### Подсчет занятий:
```
Месяц      = 8 занятий
3 месяца   = 24 занятия
Пробное    = 1 занятие
```

### Автоматическое списание:
- Занятия считаются **автоматически** по расписанию групп
- **Первое занятие** = сразу после активации/оплаты
- Каждое посещение → остаток уменьшается

### Продление раньше срока:
```javascript
Осталось: 2 занятия
Купил: Месяц (8 занятий)
Итого: 2 + 8 = 10 занятий ✅
```
**Остаток суммируется с новым абонементом!**

### Цветовая индикация в админке:
- 🟢 **>1 занятие** = зеленый (все ок)
- 🟠 **1 занятие** = оранжевый (предупреждение!)
- 🔴 **0 занятий** = красный (истек)
