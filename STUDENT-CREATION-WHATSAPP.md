## Как работает

### Процесс конвертации заявки:

```
1. Админ меняет статус заявки на "Пробное занятие"
   ↓
2. Открывается форма конвертации
   - Выбор пола
   - Выбор группы
   - Выбор типа абонемента
   ↓
3. Нажатие "СОЗДАТЬ УЧЕНИКА"
   ↓
4. POST /api/bookings/:id/convert
   ↓
5. ✅ Модальное окно с данными ученика
```

**Как работает:**
```javascript
const whatsappUrl = `https://wa.me/${phone}?text=${encodedMessage}`;
```

## Технические детали

### WhatsApp API

```javascript
https://wa.me/{phone}?text={message}
```

**Параметры:**
- `phone` - номер в формате `77001234567` (без +, скобок, пробелов)
- `text` - URL-encoded сообщение

**Пример:**
```
https://wa.me/77001234567?text=%D0%9F%D1%80%D0%B8%D0%B2%D0%B5%D1%82...
```

### Форматирование телефона

```javascript
const whatsappPhone = studentPhone.replace(/\D/g, '');
// "+7 (700) 123-45-67" → "77001234567"
```

### Кодирование сообщения

```javascript
const encodedMessage = encodeURIComponent(whatsappMessage);
// "Привет! 🎉" → "%D0%9F%D1%80%D0%B8%D0%B2%D0%B5%D1%82%21%20%F0%9F%8E%89"
```

## Шаблон сообщения

### Структура:

1. **Приветствие** - эмодзи приветствия 🎉
2. **Данные для входа** - логин и пароль с эмодзи
3. **Абонемент** - тип и количество занятий

5. **Контакты** - телефон студии
6. **Призыв** - ждем на занятиях 💃

### Эмодзи в сообщении:

- 🎉 - приветствие
- 👤 - аккаунт
- 📱 - логин
- 🔑 - пароль
- 💎 - абонемент
- 🌐 - кабинет
- 📞 - контакты
- 💃 - танцы

**Почему здесь эмодзи?**
- Это сообщение для ученика (не интерфейс)
- WhatsApp хорошо поддерживает эмодзи
- Делает сообщение дружелюбным и ярким

## Использование

### Шаг 1: Конвертация заявки
1. Вкладка "Заявки"
2. Найдите заявку со статусом "Новая"
3. Измените статус на "Пробное занятие"
4. Откроется форма конвертации

### Шаг 2: Заполнение формы
1. Выберите пол ученика
2. Выберите группу
3. Выберите тип абонемента
4. Нажмите "СОЗДАТЬ УЧЕНИКА"

### Шаг 3: Модальное окно
Появится окно с тремя кнопками:

#### Кнопка 1: "ОТПРАВИТЬ В WHATSAPP" (зеленая)
- Нажмите → откроется WhatsApp
- Сообщение уже готово
- Осталось нажать "Отправить"

#### Кнопка 2: "СКОПИРОВАТЬ СООБЩЕНИЕ" (розовая)
- Скопирует сообщение в буфер
- Можно вставить куда угодно

#### Кнопка 3: "ЗАКРЫТЬ" (серая)
- Закрывает окно
