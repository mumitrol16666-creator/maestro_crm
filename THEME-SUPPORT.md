# 🎨 Поддержка светлой/темной темы в уведомлениях

## Дата: 12 октября 2025, 20:10

## Проблема
Кастомные уведомления и диалоги использовали фиксированные цвета (`var(--black)`, `var(--white)`), которые не адаптировались под светлую/темную тему в админ-панели.

## Что сделано

### 1. Обновлен customConfirm в admin.js

**До:**
```javascript
background: var(--black);  // Всегда черный
color: var(--white);       // Всегда белый
```

**После:**
```javascript
background: var(--admin-card);    // Меняется с темой
color: var(--admin-text);         // Меняется с темой
border: 2px solid var(--admin-border);
box-shadow: 0 10px 40px var(--admin-shadow);
```

#### Используемые CSS переменные:

**Темная тема (по умолчанию):**
- `--admin-bg: #000000`
- `--admin-text: #ffffff`
- `--admin-card: #1a1a1a`
- `--admin-border: #333333`
- `--admin-shadow: rgba(0, 0, 0, 0.3)`

**Светлая тема (data-theme="light"):**
- `--admin-bg: #f5f5f5`
- `--admin-text: #000000`
- `--admin-card: #ffffff`
- `--admin-border: #e0e0e0`
- `--admin-shadow: rgba(0, 0, 0, 0.1)`

#### Overlay (фон диалога):
- Темная тема: `rgba(0, 0, 0, 0.9)` - очень темный
- Светлая тема: `rgba(0, 0, 0, 0.6)` - полупрозрачный

### 2. Обновлен showNotification в script.js

Добавлена автоопределение темы:

```javascript
// Определяем, находимся ли мы в админке
const isAdminPage = document.body.classList.contains('admin-body');
const isLightTheme = document.documentElement.getAttribute('data-theme') === 'light';

// Выбираем правильные цвета
const bgColor = isAdminPage ? 'var(--admin-card)' : 'var(--black)';
const textColor = isAdminPage ? 'var(--admin-text)' : 'var(--white)';
const shadow = isAdminPage ? 'var(--admin-shadow)' : 'rgba(0, 0, 0, 0.3)';
```

### 3. Поддержка обеих тем

#### В темной теме:
- **Фон окна:** Темно-серый (#1a1a1a)
- **Текст:** Белый (#ffffff)
- **Рамка:** Серая (#333333)
- **Overlay:** Почти черный (opacity 0.9)
- **Розовая рамка** остается яркой

#### В светлой теме:
- **Фон окна:** Белый (#ffffff)
- **Текст:** Черный (#000000)
- **Рамка:** Светло-серая (#e0e0e0)
- **Overlay:** Полупрозрачный черный (opacity 0.6)
- **Розовая рамка** остается яркой

### 4. Кнопки confirm

#### Кнопка "ДА":
- Всегда розовый фон (`var(--pink)`)
- Белый текст (`#ffffff`) - для контраста

#### Кнопка "НЕТ":
- Прозрачный фон
- Цвет текста: `var(--admin-text)` (меняется с темой)
- Рамка: `var(--admin-border)` (меняется с темой)

## Преимущества

### ✅ Автоматическая адаптация
- Уведомления автоматически подстраиваются под текущую тему
- Не нужно вручную менять стили при переключении

### ✅ Читаемость
- В светлой теме: черный текст на белом фоне
- В темной теме: белый текст на темном фоне
- Всегда хороший контраст

### ✅ Единый стиль
- Соответствует дизайну админ-панели
- Использует те же CSS переменные
- Гармонирует с остальными элементами

### ✅ Гибкость
- Легко добавить новые темы
- Просто настроить цвета
- Не нужно дублировать код

## Тестирование

### Шаг 1: Очистите кэш
```
Ctrl + Shift + R (Windows)
Cmd + Shift + R (Mac)
```

### Шаг 2: Откройте админ-панель

### Шаг 3: Проверьте в темной теме
1. Попробуйте создать/удалить запись
2. Проверьте уведомления
3. Убедитесь, что текст белый на темном фоне

### Шаг 4: Переключите на светлую тему
1. Нажмите кнопку переключения темы (обычно в хедере)
2. Попробуйте те же действия
3. Убедитесь, что текст черный на белом фоне

### Шаг 5: Проверьте профиль ученика
- В профиле (темная тема по умолчанию)
- Уведомления должны работать как раньше
- Фон черный, текст белый

## CSS переменные для других тем

Если нужно добавить новую тему, определите эти переменные:

```css
[data-theme="your-theme"] {
    --admin-bg: <цвет фона>;
    --admin-text: <цвет текста>;
    --admin-card: <цвет карточек>;
    --admin-border: <цвет рамок>;
    --admin-hover: <цвет hover>;
    --admin-shadow: <тень>;
}
```

И уведомления автоматически подстроятся!
