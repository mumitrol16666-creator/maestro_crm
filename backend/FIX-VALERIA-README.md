# Исправление дублирования платежей для ученика Валерия Валерия

## Проблема
- Показывает 33 занятия (должно быть 11)
- Две оплаты по 5000₸ (должна быть одна)
- К оплате 44 000₸ (должно быть 13 000₸)

## Должно быть
- 11 занятий
- 5000₸ аванс (один платеж)
- 1000₸ пробное занятие
- К оплате 13 000₸

## Как исправить

1. Запустите скрипт исправления:
```bash
cd backend
node fix-valeria-duplicate-payment.js
```

2. Скрипт автоматически:
   - Найдет ученика "Валерия Валерия"
   - Удалит дублирующиеся платежи по 5000₸ (оставит самый старый)
   - Исправит количество занятий (с 33 на 11)
   - Исправит остаток к оплате (с 44 000 на 13 000)
   - Покажет финальную статистику

3. После запуска проверьте результаты в админ-панели

## Защита от дублирования

Добавлена защита от дублирования платежей:

### Backend
- Проверка на дубликаты перед созданием платежа (в пределах 5 минут)
- Возвращает ошибку 409 если обнаружен дубликат

### Frontend
- Защита от двойного клика (блокировка кнопки отправки)
- Показ предупреждения при обнаружении дубликата
