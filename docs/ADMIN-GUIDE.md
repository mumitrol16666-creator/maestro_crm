## 🔥 ОСНОВНЫЕ ФУНКЦИИ:

### ✅ **Изменить статус заявки:**
1. Перейти в "Заявки"
2. Нажать кнопку "Статус" у нужной заявки
3. Статус циклично меняется: Новая → Обработано → Записался → Новая

### ✅ **Подтвердить платеж:**
1. Перейти в "Платежи"
2. Найти платеж со статусом "Ожидает"
3. Нажать "Подтвердить"
4. Платеж меняется на "Оплачен"
5. Абонемент ученика активируется

### ✅ **Поиск ученика:**
1. Перейти в "Ученики"
2. Ввести имя или телефон в поиск
3. Таблица фильтруется в реальном времени

### ✅ **Посмотреть группу:**
1. Перейти в "Группы"
2. Нажать "Ученики" на карточке группы
3. Откроется alert с информацией

## 🔌 ИНТЕГРАЦИЯ С BACKEND (Будущее):

### Когда будет backend, нужно:

**1. Заменить демо данные на API:**
```javascript
// Вместо:
const demoBookings = [...];

// Будет:
async function fetchBookings() {
    const response = await fetch('/api/admin/bookings', {
        headers: { 'Authorization': `Bearer ${token}` }
    });
    return await response.json();
}
```

**2. Подключить действия к API:**
```javascript
// Изменить статус заявки
async function changeBookingStatus(id, newStatus) {
    await fetch(`/api/admin/bookings/${id}/status`, {
        method: 'PATCH',
        headers: {
            'Authorization': `Bearer ${token}`,
            'Content-Type': 'application/json'
        },
        body: JSON.stringify({ status: newStatus })
    });
}
```

**3. Добавить авторизацию:**
```javascript
// Проверка прав доступа
async function checkAdminAccess() {
    const token = localStorage.getItem('adminToken');
    if (!token) {
        window.location.href = 'login.html';
        return;
    }
    
    // Проверка токена на сервере
    const response = await fetch('/api/auth/verify', {
        headers: { 'Authorization': `Bearer ${token}` }
    });
    
    if (!response.ok) {
        window.location.href = 'login.html';
    }
}
```
