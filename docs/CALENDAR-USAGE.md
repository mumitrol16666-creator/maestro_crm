# 📅 КАК ИСПОЛЬЗОВАТЬ КАЛЕНДАРЬ

## 🎯 Доступ

**Кто видит вкладку "Расписание":**
- ✅ Преподаватели (teacher)
- ✅ Админы (admin)
- ✅ Супер-админы (super_admin)

**Не видят:**
- ❌ Менеджеры по продажам (sales_manager)
- ❌ Ученики (student)

---

## 🚀 БЫСТРЫЙ СТАРТ

### 1. Зайдите в админку:
```
http://192.168.100.30:8000/frontend/public/admin.html
```

### 2. Войдите как преподаватель или админ

### 3. Кликните на "Расписание" в сайдбаре

---

## 📝 СОЗДАНИЕ ЗАНЯТИЯ

### Вариант 1: Кнопка "СОЗДАТЬ ЗАНЯТИЕ"
1. Нажмите кнопку сверху справа
2. Заполните форму:
   - **Группа** (обязательно)
   - **Дата** (обязательно)
   - **Время начала** (по умолчанию 18:00)
   - **Время окончания** (по умолчанию 19:30)
   - **Продолжительность** (90 минут)
   - **Заметки** (опционально)

### Вариант 2: Клик по дате в календаре
1. Кликните на нужную дату
2. Форма откроется с предзаполненной датой

---

## 🔄 ПОВТОРЯЮЩИЕСЯ ЗАНЯТИЯ

### Как создать серию занятий:

1. Поставьте галочку **"ПОВТОРЯЮЩЕЕСЯ ЗАНЯТИЕ"**
2. Выберите дни недели:
   ```
   ☑ Пн  ☐ Вт  ☑ Ср  ☐ Чт  ☑ Пт  ☐ Сб  ☐ Вс
   ```
3. Укажите **дату окончания повторений**
   - По умолчанию: +3 месяца

### Пример:
```
Группа: K-pop Начинающие
Дата начала: 15 октября 2025
Время: 18:00 - 19:30
Дни недели: Понедельник, Среда
Дата окончания: 15 января 2026

Результат: Будет создано ~26 занятий
```

---

## 🖱️ ПЕРЕНОС ЗАНЯТИЯ (Drag & Drop)

### Перенести на другую дату:
1. **Захватите** мышкой блок занятия
2. **Перетащите** на новую дату
3. **Отпустите** мышку
4. ✅ Занятие автоматически сохранится с новой датой

**Важно:** Время останется прежним!

---

## 📊 ВИДЫ КАЛЕНДАРЯ

### Переключение вида:
- **Месяц** - полный месяц, удобно для планирования
- **Неделя** - детальный вид с часами
- **День** - почасовой график на один день

### Навигация:
- **← →** - предыдущий/следующий период
- **Сегодня** - вернуться к текущей дате

---

## 🎨 ЧТО ПОКАЗЫВАЕТ КАЛЕНДАРЬ

### Блок занятия:
```
┌─────────────────────────┐
│ K-pop Начинающие (8)    │
│ 18:00 - 19:30           │
└─────────────────────────┘
```

- **Название**: Группа + Направление
- **(8)**: Количество отметок посещаемости
- **Время**: Начало и конец

### Цвета:
- 🟣 **Розовый** - Обычное занятие
- ✨ **При наведении** - Светлеет

---

## 🔍 ЧТО ВИДЯТ РАЗНЫЕ РОЛИ

### Преподаватель:
- Видит **только свои группы**
- Может создавать занятия **только для своих групп**
- Может переносить **только свои занятия**

### Админ / Супер-Админ:
- Видят **все занятия всех преподавателей**
- Могут создавать занятия **для любых групп**
- Могут переносить **любые занятия**

---

## ⚙️ ТЕХНИЧЕСКИЕ ДЕТАЛИ

### Автоматическое обновление:
- При создании занятия → календарь обновляется
- При переносе → сохраняется в БД моментально
- При переключении месяца → загружаются занятия

### API Endpoints используемые:
```
GET  /api/classes?start=2025-10-01&end=2025-10-31
POST /api/classes
PATCH /api/classes/:id
```

## 🐛 TROUBLESHOOTING

### Календарь не загружается:
1. Проверьте что backend запущен:
   ```bash
   curl http://localhost:5001/
   ```
2. Откройте консоль браузера (F12)
3. Проверьте ошибки

### Занятия не показываются:
1. Проверьте что у вас есть группы
2. Создайте тестовое занятие
3. Переключите вид календаря (месяц/неделя)

### Drag & Drop не работает:
1. Убедитесь что вы авторизованы
2. Проверьте права (преподаватель/админ)
3. Проверьте что занятие принадлежит вам
