# 🎯 Управление Направлениями

## Описание

Модуль управления направлениями позволяет супер-администратору динамически добавлять, редактировать и удалять танцевальные направления студии.

---

## 🔐 Доступ

**Только для Super Admin!**

Вкладка "Направления" видна только пользователям с ролью `super_admin`.

---

## ✨ Функционал

### 1. Просмотр направлений
- Таблица со всеми направлениями
- Отображение порядка сортировки
- Статус (Активно/Неактивно)

### 2. Добавление направления
- Название направления (уникальное)
- Порядок отображения (число)
- Автоматическая активация

### 3. Редактирование направления
- Изменение названия
- Изменение порядка
- Изменение статуса

### 4. Удаление направления
- ⚠️ Невозможно удалить направление, если оно используется в группах
- Подтверждение удаления

---

## 🗄️ База данных

### Модель Direction

```javascript
{
  name: String (unique),      // Название направления
  description: String,        // Описание направления
  minAge: Number,             // Минимальный возраст
  level: String,              // Уровень подготовки
  image: String,              // URL изображения (опционально)
  isActive: Boolean,          // Активно/неактивно
  order: Number,              // Порядок отображения
  createdBy: ObjectId,        // ID создателя
  createdAt: Date,            // Дата создания
  updatedAt: Date             // Дата обновления
}
```

## 🔌 API Endpoints

### GET `/api/directions/public`
- **Доступ:** Публичный (без авторизации)
- **Описание:** Получить активные направления для отображения на главной странице
- **Ответ:** Список активных направлений с полями: `name`, `description`, `minAge`, `level`, `image`, `order`

### GET `/api/directions`
- **Доступ:** Все авторизованные
- **Описание:** Получить все направления (для админки)
- **Ответ:** Полный список направлений, отсортированных по `order` и `name`

### POST `/api/directions`
- **Доступ:** Super Admin
- **Описание:** Создать новое направление
- **Body:**
  ```json
  {
    "name": "K-pop",
    "description": "Энергичный корейский поп-данс с яркой хореографией",
    "minAge": 10,
    "level": "Все уровни",
    "image": "https://example.com/kpop.jpg",
    "order": 1
  }
  ```

### PATCH `/api/directions/:id`
- **Доступ:** Super Admin
- **Описание:** Обновить направление
- **Body:**
  ```json
  {
    "name": "K-pop Updated",
    "description": "Обновленное описание",
    "minAge": 12,
    "level": "Продвинутый",
    "image": "https://example.com/new-image.jpg",
    "order": 2,
    "isActive": true
  }
  ```

### DELETE `/api/directions/:id`
- **Доступ:** Super Admin
- **Описание:** Удалить направление
- **Проверка:** Направление не должно использоваться в группах

## ⚙️ Интеграция с другими модулями

### Группы
- При создании/редактировании группы выбирается направление из списка
- Направление нельзя удалить, если оно используется в активных группах

### Преподаватели
- При создании преподавателя выбираются направления, которые он ведет
- Список направлений обновляется динамически

### Заявки
- При создании заявки выбирается направление
- Отображается в списке заявок

## 📝 Примечания

1. Названия направлений должны быть уникальными
2. Порядок отображения определяет сортировку (меньше = выше)
3. Неактивные направления можно скрыть в UI (опционально)
4. При удалении направления проверяется его использование в группах
5. История изменений сохраняется через `createdAt` и `updatedAt`

## 🐛 Возможные ошибки

### "Направление с таким названием уже существует"
- Проверьте уникальность названия
- Учитывается точное совпадение с учетом регистра

### "Невозможно удалить направление. Оно используется в X группах"
- Удалите или измените направление в группах
- Затем повторите попытку удаления

### "Ошибка при получении направлений"
- Проверьте подключение к БД
- Проверьте наличие токена авторизации
