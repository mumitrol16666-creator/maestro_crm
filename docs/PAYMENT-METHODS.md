## 📱 МЕТОД 1: KASPI ПЕРЕВОДЫ (РЕКОМЕНДУЮ ДЛЯ СТАРТА)

### Как работает:
```
1. Ученик выбирает абонемент на сайте
2. Видит инструкцию:
   
   ┌──────────────────────────────────┐
   │ Переведите 22000₸ на Kaspi:     │
   │ +7 (700) 095-09-04              │
   │                                  │
   │ В комментарии укажите ваше имя   │
   │ После оплаты напишите в Telegram │
   └──────────────────────────────────┘

3. Ученик переводит деньги
4. Пишет вам в Telegram: "Оплатил абонемент"
5. Вы заходите в админку
6. Нажимаете "Подтвердить оплату"
7. Абонемент автоматически активируется
```

### Преимущества:
✅ Работает сразу (0 времени на настройку)  
✅ Без комиссий (0%)  
✅ Не нужен ИП/ТОО  
✅ Привычно для казахстанцев  

### Недостатки:
❌ Ручная работа (проверить Kaspi + подтвердить)  
❌ Задержка активации (до проверки админом)  

### Код в админке:
```javascript
// Кнопка "Подтвердить Kaspi оплату"
confirmKaspiPayment(studentId, amount) {
  // 1. Создается Payment запись
  // 2. Membership активируется
  // 3. Уведомление ученику в Telegram
  // 4. Генерируется чек (PDF)
}
```

## 🚀 ФАЗА 2 (Месяц 4+): АВТОМАТИЗАЦИЯ

### МЕТОД 3: KASPI PAY API (АВТОМАТИЧЕСКИЙ)

#### Что нужно для подключения:

**Шаг 1: Регистрация бизнеса**
1. Открыть **business.kaspi.kz**
2. Зарегистрироваться как ИП или ТОО
3. Документы:
   - Свидетельство о регистрации ИП/ТОО
   - БИН (бизнес идентификационный номер)
   - Паспорт
   - Контактные данные
4. Время одобрения: 1-5 рабочих дней

**Шаг 2: Подключение Kaspi Pay**
1. В личном кабинете → Сервисы → **Kaspi Pay**
2. Заполнить заявку:
   - URL сайта
   - Описание бизнеса
   - Callback URL для webhook: `https://ваш-сайт.kz/api/payments/kaspi-webhook`
3. Документы:
   - Выписка из банка
   - Лицензии (если нужны)
4. Время одобрения: 1-2 недели
5. Получаете:
   - **Client ID**
   - **Client Secret**
   - **Merchant ID**

**Шаг 3: Тестирование**
1. Kaspi даст тестовый доступ
2. Тестовые платежи
3. После проверки - боевой доступ

#### Как работает после подключения:
```
1. Ученик выбирает абонемент
2. Нажимает "Оплатить через Kaspi"
3. Показывается QR код (генерится автоматически)
4. Ученик сканирует → оплачивает
5. Kaspi отправляет webhook на ваш сервер
6. Абонемент активируется АВТОМАТИЧЕСКИ
7. Ученик получает уведомление в Telegram
8. Чек отправляется на email
```

#### Преимущества:
✅ Автоматизация (0 ручной работы)  
✅ Мгновенная активация  
✅ Профессиональный вид  
✅ Меньше ошибок  
✅ Удобно для учеников  

#### Недостатки:
❌ Нужен ИП/ТОО (время на оформление)  
❌ Комиссия 1-2%  
❌ Время на одобрение (2-3 недели)  
❌ Интеграция API (2-3 дня разработки)  

#### Стоимость:
```
Комиссия Kaspi Pay: 1-2% от суммы
Пример:
- Абонемент 22000₸ → комиссия ~220-440₸
- В месяц 20 абонементов → ~4400-8800₸ комиссии
```

## 🔐 БЕЗОПАСНОСТЬ ПЛАТЕЖЕЙ:

### Что важно:
```
✅ HTTPS обязательно (SSL сертификат)
✅ Webhook signature verification (проверка подписи от Kaspi)
✅ Не хранить данные карт (PCI DSS compliance)
✅ Логирование всех транзакций
✅ Двойная проверка суммы
✅ Защита от повторных оплат (idempotency)
```

### Пример проверки Kaspi webhook:
```javascript
function verifyKaspiSignature(req) {
  const signature = req.headers['x-kaspi-signature'];
  const payload = JSON.stringify(req.body);
  
  const expectedSignature = crypto
    .createHmac('sha256', KASPI_SECRET_KEY)
    .update(payload)
    .digest('hex');
  
  return signature === expectedSignature;
}
```

// 1. Создание платежа

  "orderId": "SOD-20251008-001",

  "amount": 22000,
  "timestamp": "2025-10-08T16:30:00Z",
  "signature": "a3d5f8e9..." // Для проверки подлинности

// 5. Ваш сервер:
