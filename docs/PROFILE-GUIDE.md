## 🔐 КАК ВОЙТИ (ДЕМО РЕЖИМ):

1. Нажать на кнопку **"ПРОФИЛЬ"** в хедере

- **Полное имя**
- **Телефон** (автоформат: +7 (XXX) XXX-XX-XX)
- **Email** (опционально)
- **Пароль** (минимум 6 символов)
- **Повторите пароль** (должны совпадать)

## 💾 КАК РАБОТАЕТ (Демо режим):

### LocalStorage хранит:
```javascript
isLoggedIn: 'true'       // Статус авторизации
userName: 'Имя'          // Имя пользователя
userPhone: '+7 (XXX)'    // Телефон
userEmail: 'email@'      // Email
```

### При входе/регистрации:
- Проверка минимальных требований
- Сохранение данных в localStorage
- Перенаправление на profile.html

### При открытии профиля:
- Проверка `isLoggedIn`
- Если `false` → редирект на login.html
- Если `true` → загрузка данных из localStorage

### При выходе:
- Очистка всех данных из localStorage
- Редирект на login.html

## 🔌 ИНТЕГРАЦИЯ С BACKEND (Будущее):

### Что нужно подключить:

#### 1. Авторизация (login.html):
```javascript
POST /api/auth/login
Body: { phone, password }
Response: { token, user }
```

#### 2. Регистрация (register.html):
```javascript
POST /api/auth/register
Body: { name, phone, email, password }
Response: { token, user }
```

#### 3. Загрузка профиля (profile.html):
```javascript
GET /api/profile
Headers: { Authorization: 'Bearer TOKEN' }
Response: {
  user: { name, phone, email },
  groups: [...],
  membership: {...},
  upcomingClasses: [...],
  practices: [...],
  attendance: [...]
}
```

#### 4. Заморозка занятия:
```javascript
POST /api/memberships/freeze
Headers: { Authorization: 'Bearer TOKEN' }
Body: { groupId, date, reason }
Response: { success, freezesLeft }
```

#### 5. Запись на практику:
```javascript
POST /api/practices/:id/attend
Headers: { Authorization: 'Bearer TOKEN' }
Response: { success, enrolled }
```
