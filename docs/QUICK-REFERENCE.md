## 🔑 КЛЮЧЕВЫЕ ОСОБЕННОСТИ СИСТЕМЫ:

### 📋 Заявки:
- 4 статуса: new → processed → enrolled/rejected
- Админ может создавать заявки сам
- Автоматические уведомления в Telegram

### 👥 Группы:
- Разные группы для каждого направления
- У каждой группы свое расписание
- Макс кол-во учеников в группе
- Ученик может быть в 2 группах одновременно

### 💃 Практики:
- Общие для всех направлений
- Запись на практику из личного кабинета
- Отметка посещения

### 🎫 Абонементы:
- Типы: пробное (2000₸), месяц (22000₸), 3 месяца (55000₸)
- Оплата помесячно
- Заморозка: 2 занятия в месяц
- Автоматическое истечение

### 💳 Оплата:
- ФАЗА 1: Kaspi переводы + наличные (ручное подтверждение)
- ФАЗА 2: Kaspi Pay API (автоматически)

## 🚀 ЗАПУСК РАЗРАБОТКИ:

### Когда готовы начать:

```bash
# 1. Создать структуру
mkdir -p backend/src/{config,models,routes,controllers,middleware,utils}

# 2. Инициализировать npm
cd backend
npm init -y

# 3. Установить зависимости
npm install express mongoose cors dotenv bcryptjs jsonwebtoken express-validator helmet morgan axios

# 4. Создать MongoDB Atlas кластер
# (через веб-интерфейс mongodb.com/atlas)

# 5. Запустить сервер
npm start
```
