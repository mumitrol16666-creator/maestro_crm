# 🤖 НАСТРОЙКА TELEGRAM БОТА

## 📱 ШАГ 1: СОЗДАНИЕ БОТА

### 1. Откройте Telegram
Найдите **@BotFather** (официальный бот для создания ботов)

### 2. Создайте нового бота
Отправьте команду:
```
/newbot
```

### 3. Введите имя бота
```
Sense of Dance Bot
```
(Это имя будет видно пользователям)

### 4. Введите username бота
```
senseofdance_bot
```
(Должен заканчиваться на `bot` и быть уникальным)

Если занято, попробуйте:
- `senseofdance_aktobe_bot`
- `sod_aktobe_bot`
- `sense_dance_bot`

### 5. Скопируйте TOKEN
BotFather отправит сообщение с токеном:
```
Use this token to access the HTTP API:
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

⚠️ **НЕ ПОКАЗЫВАЙТЕ ТОКЕН НИКОМУ!**

---

## 🆔 ШАГ 2: ПОЛУЧЕНИЕ CHAT_ID

### Вариант 1: Через @userinfobot
1. Найдите бота **@userinfobot** в Telegram
2. Отправьте ему `/start` или любое сообщение
3. Он пришлет ваш ID (например: `123456789`)
4. Скопируйте это число

### Вариант 2: Через @getmyid_bot
1. Найдите бота **@getmyid_bot**
2. Нажмите Start
3. Скопируйте ваш Chat ID

---

## ⚙️ ШАГ 3: НАСТРОЙКА КОДА

Откройте файл **`telegram-config.js`** и замените:

```javascript
const TELEGRAM_CONFIG = {
    // Вставьте токен из BotFather
    BOT_TOKEN: '1234567890:ABCdefGHIjklMNOpqrsTUVwxyz',
    
    // Вставьте ваш Chat ID
    CHAT_ID: '123456789',
    
    // Остальное не трогайте
    MESSAGE_TEMPLATE: {
        emoji: '💃',
        title: 'НОВАЯ ЗАЯВКА | SENSE OF DANCE',
        separator: '━━━━━━━━━━━━━━━━━━━━'
    }
};
```

## 🔒 БЕЗОПАСНОСТЬ

### ⚠️ ВАЖНО:

1. **Не публикуйте TOKEN** в открытом доступе
2. **Не коммитьте** `telegram-config.js` в GitHub
3. **Добавьте в .gitignore:**
```bash
echo "telegram-config.js" >> .gitignore
```

### Для продакшена:
Используйте переменные окружения вместо хранения токена в файле.

## 🐛 РЕШЕНИЕ ПРОБЛЕМ

### Заявка не приходит?

**1. Проверьте консоль браузера (F12)**
- Есть ошибки?
- Что пишет в консоли?

**2. Проверьте токен и chat_id**
```javascript
// Должно быть БЕЗ пробелов и кавычек
BOT_TOKEN: '1234567890:ABC...',  ✅
BOT_TOKEN: ' 1234567890:ABC... ', ❌
```

**3. Убедитесь что бот запущен**
- Напишите боту `/start`
- Бот должен ответить

**4. Проверьте интернет**
- Telegram API работает?
- Есть доступ к api.telegram.org?
