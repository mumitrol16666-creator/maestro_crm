# 👥 СИСТЕМА РОЛЕЙ - SENSE OF DANCE

## 🎯 КРАТКАЯ СПРАВКА

### 5 УРОВНЕЙ ДОСТУПА:

---

## 1️⃣ SUPER ADMIN (Владелец)
**Количество:** 1 человек
**Доступ:** Абсолютно всё

### Может:
✅ Всё что может Admin
✅ **Управлять администраторами:**
   - Создавать новых Admin
   - Удалять Admin
   - Повышать/понижать роли
✅ **Управлять преподавателями:**
   - Создавать
   - Редактировать
   - **Удалять** (только Super Admin!)
✅ Доступ к системным настройкам
✅ Просмотр логов действий

### Видит в админке:
- 📊 Дашборд
- 📝 Заявки
- 👥 Ученики
- 👨‍💼 Преподаватели (полное управление)
- 🔑 **Администраторы** ⭐
- 👥 Группы
- 💳 Абонементы
- 🎯 Практики
- 💰 Платежи
- 📈 Аналитика
- ⚙️ **Настройки** ⭐

---

## 2️⃣ ADMIN (Менеджер)
**Количество:** Несколько человек
**Доступ:** Почти всё (кроме управления админами)

### Может:
✅ Управлять заявками (все статусы)
✅ Управлять учениками (создавать, редактировать, удалять)
✅ Управлять группами (создавать, редактировать, удалять)
✅ Управлять абонементами (создавать, продлевать)
✅ Управлять практиками
✅ Подтверждать платежи
✅ **Создавать преподавателей**
✅ **Редактировать преподавателей**
✅ Назначать группы преподавателям
✅ Видеть всю статистику

### НЕ может:
❌ Создавать других Admin
❌ Удалять Admin
❌ Изменять роли на Admin
❌ Удалять преподавателей (только Super Admin)
❌ Доступ к системным настройкам

### Видит в админке:
- 📊 Дашборд
- 📝 Заявки
- 👥 Ученики
- 👨‍💼 Преподаватели (создание/редактирование)
- 👥 Группы
- 💳 Абонементы
- 🎯 Практики
- 💰 Платежи
- 📈 Аналитика

---

## 3️⃣ SALES MANAGER (Менеджер по продажам)
**Количество:** Несколько человек
**Доступ:** Заявки + контакты учеников (без финансов)

### Может:
✅ **Работать с заявками:**
   - Просматривать все заявки
   - Изменять статусы
   - Создавать заявки
   - Конвертировать в учеников
✅ **Консультировать клиентов:**
   - Видеть список всех групп
   - Видеть расписание
   - Знать количество свободных мест
✅ **Работать с учениками:**
   - Видеть контакты учеников
   - Видеть статус абонемента (когда истекает)
   - Звонить о продлении
✅ **Создавать пробные занятия:**
   - Записать на пробное
   - Добавить в группу

### НЕ может:
❌ Видеть платежи и финансы
❌ Подтверждать оплаты
❌ Создавать полные абонементы (месяц/3 месяца)
❌ Удалять учеников
❌ Создавать/редактировать группы
❌ Управлять преподавателями
❌ Видеть аналитику

### Видит в панели:
- 📝 Заявки (полное управление)
- 👥 Ученики (контакты + статус абонемента)
- 👥 Группы (просмотр для консультаций)
- 🎫 Пробные занятия

---

## 4️⃣ TEACHER (Преподаватель)
**Количество:** Несколько человек
**Доступ:** Только свои группы

### Может:
✅ Видеть **только свои группы**
✅ Видеть расписание своих групп
✅ Видеть учеников **только из своих групп**
✅ Отмечать посещаемость в своих группах
✅ Просматривать историю посещений
✅ Видеть практики по своему направлению
✅ Видеть список записавшихся на практики

### НЕ может:
❌ Видеть другие группы
❌ Видеть всех учеников студии
❌ Создавать/редактировать группы
❌ Управлять абонементами
❌ Видеть платежи
❌ Видеть заявки
❌ Видеть финансы
❌ Создавать учеников

### Видит в панели:
- 📊 Мои группы
- 📅 Расписание
- ✅ Посещаемость (своих групп)
- 👥 Ученики (только из моих групп)
- 🎯 Практики (мои направления)

---

## 5️⃣ STUDENT (Ученик)
**Количество:** Неограниченно
**Доступ:** Только свой профиль

### Может:
✅ Видеть свой профиль
✅ Редактировать свои данные
✅ Видеть свои группы (до 2 групп)
✅ Видеть расписание
✅ Видеть свой абонемент
✅ Замораживать занятия (2 в месяц)
✅ Записываться на практики
✅ Видеть историю посещений
✅ Видеть историю платежей

### НЕ может:
❌ Видеть других учеников
❌ Доступ к админке
❌ Видеть финансы
❌ Менять свою роль

### Видит в кабинете:
- 👤 Мой профиль
- 👥 Мои группы
- 💳 Мой абонемент
- 🎯 Практики
- 📜 История посещений

## 🎨 КАК СОЗДАВАТЬ ПОЛЬЗОВАТЕЛЕЙ

### 1. Создание ученика (Admin/Super Admin):
```
Кнопка: [Добавить ученика]
Поля:
- Имя ✅
- Телефон ✅
- Email (опционально)
- Дата рождения (опционально)
- Пароль: автогенерация ✅

Роль: Student (автоматически)
```

### 2. Создание преподавателя (Admin/Super Admin):
```
Кнопка: [Добавить преподавателя]
Поля:
- Имя ✅
- Телефон ✅
- Email
- Направления: [☐ K-pop] [☐ Bachata] ...
- Био
- Фото
- Пароль: автогенерация ✅

Роль: Teacher (автоматически)
```

### 3. Создание менеджера по продажам (Admin/Super Admin):
```
Кнопка: [Добавить менеджера]
Поля:
- Имя ✅
- Телефон ✅
- Email
- Пароль: автогенерация ✅

Роль: Sales Manager (автоматически)

ℹ️ Менеджер сможет работать с заявками и консультировать клиентов
```

### 4. Создание админа (только Super Admin):
```
Кнопка: [Добавить администратора] ⭐
Поля:
- Имя ✅
- Телефон ✅
- Email
- Пароль: автогенерация ✅

Роль: Admin (автоматически)

⚠️ Предупреждение перед созданием
```

### 5. Изменение роли пользователя:
```
В карточке пользователя:

Роль: [▼ Student / Sales Manager / Teacher / Admin]

При изменении на Sales Manager (Admin/Super Admin):
ℹ️ Подтверждение + уведомление в Telegram

При изменении на Teacher (Admin/Super Admin):
ℹ️ Подтверждение + уведомление в Telegram

При изменении на Admin (только Super Admin):
⚠️ Подтверждение + уведомление в Telegram
```

## 🛡️ ПРАВИЛА БЕЗОПАСНОСТИ

### Super Admin:
- ❌ Не может удалить себя
- ❌ Не может изменить свою роль
- ✅ Может делать всё остальное

### Admin:
- ❌ Не может создать другого Admin
- ❌ Не может удалить Admin
- ❌ Не может изменить роль на Admin
- ❌ Не может удалить Teacher
- ❌ Не может изменить свою роль
- ✅ Может создавать Teacher

### Sales Manager:
- ❌ Не может видеть финансы
- ❌ Не может подтверждать оплаты
- ❌ Не может создавать полные абонементы
- ❌ Не может удалять учеников
- ❌ Не может создавать группы
- ✅ Может работать с заявками
- ✅ Может создавать пробные занятия

### Teacher:
- ❌ Видит только свои группы
- ❌ Не может создавать пользователей
- ❌ Не может видеть финансы
- ✅ Может отмечать посещаемость

### Student:
- ❌ Не может видеть других учеников
- ❌ Не может менять свою роль
- ❌ Нет доступа к админке
- ✅ Может управлять своим профилем

## 📋 API ENDPOINTS

### Управление администраторами:
```
GET    /api/admins              [Super Admin]
POST   /api/admins              [Super Admin]
DELETE /api/admins/:id          [Super Admin]
PATCH  /api/admins/:id/demote   [Super Admin]
```

### Управление менеджерами по продажам:
```
GET    /api/sales-managers              [Super Admin / Admin]
POST   /api/sales-managers              [Super Admin / Admin]
DELETE /api/sales-managers/:id          [Super Admin / Admin]
PATCH  /api/sales-managers/:id/demote   [Super Admin / Admin]
```

### Управление преподавателями:
```
GET    /api/teachers                      [Admin]
POST   /api/teachers                      [Admin]
PATCH  /api/teachers/:id                  [Admin]
DELETE /api/teachers/:id                  [Super Admin]
POST   /api/teachers/:id/assign-group     [Admin]
DELETE /api/teachers/:id/remove-group     [Admin]
```

### Управление учениками:
```
POST   /api/students                    [Admin]
GET    /api/students                    [Admin]
GET    /api/students/:id                [Auth]
PATCH  /api/students/:id                [Auth]
DELETE /api/students/:id                [Admin]
POST   /api/students/:id/change-role    [Super Admin]
```
