# WhatsApp-напоминания

## Назначение

Отдельная рабочая очередь для ролей `admin` и `super_admin`. CRM не отправляет сообщения автоматически: администратор нажимает кнопку, после чего открывается WhatsApp-чат ученика с заранее заполненным текстом.

## Очереди

1. Ученики, у которых ещё предстоит урок сегодня.
2. Ученики, у которых урок завтра.
3. Ученики с активным непробным абонементом, где осталось ровно одно занятие.
4. Запланированные контакты из очереди оплат и продлений (`followUpAt`, статус не `closed`).

## Источники данных

- Уроки: `Class`, индивидуальный ученик, участники занятия и активный состав группы.
- Продление: `Membership.classesRemaining`.
- Задачи: существующие поля `Membership.followUpStatus` и `Membership.followUpAt`.

Получатели внутри одного урока дедуплицируются по паре `classId + studentId`.

## WhatsApp

Используется официальный deeplink `https://wa.me/<phone>?text=<message>`. Сообщение только подставляется в поле ввода, окончательную отправку подтверждает администратор.

Ссылка на оплату вынесена в `WHATSAPP_PAYMENT_LINK` в модуле напоминаний. Пока ссылка не задана, клиенту предлагается запросить её в чате.
