## 🎮 КАК ИСПОЛЬЗОВАТЬ

### Открыть систему:
```
Админка → Sidebar → "Управление ролями"
```

### Изменить функциональное право:
```
1. Таблица "ФУНКЦИОНАЛЬНЫЕ ПРАВА"
2. Найти функцию (например, "Управление абонементами")
3. Кликнуть на ячейку нужной роли
4. ON → OFF или OFF → ON
5. Автоматическое сохранение
```

### Изменить видимость раздела:
```
1. Таблица "ВИДИМОСТЬ РАЗДЕЛОВ В АДМИНКЕ"
2. Найти раздел (например, "Практики")
3. Кликнуть на ячейку роли
4. ON → OFF (раздел скроется из sidebar для этой роли)
```

### Сбросить к дефолтным:
```
1. Кнопка "Сбросить к дефолтным" внизу
2. Подтвердить
3. Все роли получат стандартные права
```

## 🎨 ИНТЕРФЕЙС

### Цвета статусов:
- 🟢 **ON** - Зеленый (право включено)
- ⚪ **OFF** - Серый (право выключено)
- 🟡 **LOCKED** - Желтый (Super Admin, нельзя менять)

### Hover эффекты:
- Наведение на **ON** → Краснеет (показывает что выключится)
- Наведение на **OFF** → Зеленеет (показывает что включится)
- **LOCKED** - Без hover (нельзя менять)

### Визуальный стиль:
- Без эмодзи (только текст ON/OFF)
- Прозрачный фон
- Тонкие рамки
- Верхняя розовая полоса
- В стиле остального сайта

## 💾 BACKEND

### Новая модель: `RolePermissions.js`

```javascript
{
    role: 'admin',
    permissions: {
        manageBookings: true,
        manageStudents: true,
        viewStudents: true,
        // ... всего 17 прав
    },
    visibility: {
        dashboard: true,
        bookings: true,
        students: true,
        // ... всего 10 разделов
    }
}
```

### API Роуты: `/api/permissions`

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/permissions` | Получить права всех ролей |
| PATCH | `/api/permissions/:role` | Обновить права роли |
| POST | `/api/permissions/:role/reset` | Сбросить к дефолтным |

### Дефолтные права создаются автоматически:
- При первом запросе к `/api/permissions`
- Если прав для роли нет в базе
- Через метод `getDefaultPermissions(role)`

## 💻 FRONTEND

### JavaScript функции:

| Функция | Описание |
|---------|----------|
| `loadRolesData()` | Загрузка прав с сервера |
| `renderPermissionsTable()` | Отрисовка таблицы функций |
| `renderVisibilityTable()` | Отрисовка таблицы видимости |
| `togglePermission()` | Переключение права + автосохранение |
| `resetPermissionsToDefault()` | Сброс всех прав |
| `applySidebarVisibility()` | Применение прав при загрузке |
| `initUserManagementFallback()` | Резервная логика |

### Применение прав при загрузке:

```javascript
// При инициализации админки:
initUserManagement() 
  → applySidebarVisibility()
    → Загрузка прав из API
    → Применение к sidebar
    → Показ/скрытие разделов
```
